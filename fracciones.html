FRACCIONES

Una fracción es una forma de representar una cantidad dividida en partes. Expresa un cociente no resuelto entre dos números: el numerador, que indica cuántas partes se toman, y el denominador, que señala en cuántas partes iguales se divide la unidad. Existen distintos tipos de fracciones, como las *fracciones comunes, mixtas y decimales. Las fracciones comunes se representan con una barra (horizontal u oblicua) entre el numerador y el denominador.

Ejemplo: Suma de Fracciones

La suma de fracciones con diferente denominador consiste en unir fracciones que tienen distintos valores en la parte inferior (denominadores). Como no representan partes iguales, primero se deben convertir a fracciones equivalentes con el mismo denominador. Este tipo de fracciones se llaman heterogéneas y para sumarlas se siguen estos pasos: Buscar el mínimo común denominador (MCD). Convertir las fracciones a otras equivalentes con ese denominador. Sumar los numeradores. Mantener el denominador. Simplificar si es posible.




Ejemplo de Suma de Fracciones con Diferente Denominador:

Operación:

2/3 + 1/4

Paso 1: Buscar el mínimo común denominador (MCD) entre 3 y 4 → 12

Paso 2: Convertir las fracciones:

  • 2/3 = 8/12
  • 1/4 = 3/12

Paso 3: Sumar los numeradores:

8/12 + 3/12 = 11/12

Resultado: 11/12



3. Resta de fracciones con mismo denominador

Cuando dos fracciones tienen el mismo denominador, se pueden restar directamente* restando los numeradores y conservando el denominador. Esto se debe a que están divididas en partes iguales, por lo que solo cambia la cantidad de partes (el numerador).




4. Resta de fracciones con diferente denominador

Para restar fracciones con diferente denominador, primero debes convertirlas a fracciones equivalentes con el mismo denominador. Esto se hace hallando el **mínimo común denominador (MCD)*. Luego, se restan los numeradores y se conserva el denominador común.


5. Multiplicacion de fracciones

Para multiplicar fracciones, se multiplican los numeradores entre sí y los denominadores entre sí. No es necesario que tengan el mismo denominador. El resultado se puede simplificar si es posible.




6. Dividir fracciones

Para dividir fracciones, se multiplica la primera fracción por el inverso (recíproco) de la segunda. El recíproco de una fracción se obtiene invirtiendo su numerador y denominador.
